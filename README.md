# Data-Analyst-End-to-End-Project

- This case study "QuickPay Fintech Operations" focuses on analyzing payment transaction data to evaluate merchant performance and financial risk. The primary goal was to transform raw, fragmented transaction logs into actionable business insights regarding revenue, disputes (chargebacks), and regional risk profiles.
- SQL and Python workflows for cleaning transaction data, merchant‑level reporting, payment reconciliation, JSON normalization, and dashboard creation. Designed to support business monitoring, resolve mismatches, and strengthen risk insights for QuickPay’s digital payment operations.

- Tools used :
  - Ms Excel
  - Google Colab
  - MySQL
  - Looker Studio
